# win32asm_usb_tool
win32asm可移动磁盘病毒专杀工具

## 主要文件功能简述

| **文件名**   | **依赖**     | **主要功能描述**                         |
| ------------ | ------------ | ---------------------------------------- |
| USB_Tool.Inc |              | 头文件、导入库  宏、常量、全局变量定义等 |
| myFunc.Asm   | USB_Tool.Inc | 自定义功能函数                           |
| USB_Tool.Asm | myFunc.Asm   | 主程序界面、核心功能实现等               |
